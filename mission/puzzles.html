# 开封解谜

解谜系统

在繁华的开封城内，名流云集，各具特色。其中，有西天取经归来的猪八戒，有以善行著称的相府夫妇，有高贵典雅的高府大小姐香兰，还有殷开山之女玄奘之母，以及玄奘之父陈光蕊——大阐都纲御赐状元。他们各自忙碌于慈善、寻宝、求签、祭祖等事务，共同构成了开封城七大解谜系统，这些系统既独立又相互依存，为玩家提供了丰富的探索与挑战机会。

解谜分类

  1. 天蓬帅府：猪八戒自西天归来后，受唐太宗赏识，特赐在开封设府。此府专注于饮食慈善，救济饥民，誓言要集遍世上美食，让唐人前来品尝。
  2. 御相府：相良因救唐太宗之恩，被御准在开封收集武器佳品进贡皇宫。相公亲自坐阵兵器场，确保兵器质量上乘。
  3. 御相府：相良之妻则负责征集各类护具盔甲，进贡皇宫或送至京城进行鉴定与改进。
  4. 高府：因高家与猪家有姻缘，且三藏受高家之恩，故被赐在开封设府。高府大小姐香兰在此致力于收集衣物，捐给穷苦之人。
  5. 水陆大会：普渡场里，殷夫人凭求签知世人有难，玩家可前去询问，助人解难。
  6. 水陆大会：祭贤场里，三藏之父承恩专事祭祖求贤，玩家可前往相助，共招天下贤士。
  7. 水陆大会：遏恶场里，八十万御林都督胡敬德为大会做法，誓言斩尽天下妖魔鬼怪。玩家可请其算卦，并助其斩妖除怪。

解谜规则

  1. 玩家可自由拜访七大名人，协助他们完成美食、武器、盔甲、衣物等征集任务，或参与救济、寻访、斩妖等活动，助力水陆法事。
  2. 玩家可同时进行多项任务，最多可达七项，且解谜无时间限制。完成后需回访相应府邸。
  3. 完成使命并获得认可后，玩家的品德将得到提升。
  4. 品德提升后，玩家可赴京进宫请赏，获得丰厚奖励。
  5. 若玩家遇到困难，可前往清心宫、宁心宫、静心宫三拜，每宫分“清”“宁”“静”三步。拜完后可接受新的使命。

奖励

品德提升后，玩家可立刻赴京请赏。唐太宗将听取大臣建议，赐予玩家以下御赏之一：

  1. 大臣段志贤建议赏玩家潜能；
  2. 大臣徐茂功建议赏玩家道行；
  3. 大臣杜如晦建议赏玩家技能；
  4. 大臣张士衡建议赏玩家天赋；
  5. 大臣孟子如建议赏玩家白银。
