# 【叛师绝招】

# 简介

自古有大毅力、大智慧者，涉猎广泛，辗转于多个派别之间，习
得各类庞杂武功法术，最终融会贯通，领悟出一等一的绝招，只
是此途甚是艰险，且由于所学驳杂，却又无法习得门派最精髓之
绝学了，正是鱼与熊掌不可兼得。

# 领悟条件

要修习叛师绝招，必须学会多种同类技能，且达到最低级别要求
然后需从泾河向东出海，抵达神秘岛屿，在其上领悟方可使用。
随着所学技能数量与平均级别提升，绝招威力也会有巨大提升。
绝招所需技能达到350为Level1,500级为Level2,650级为Level3

# 级别要求

  • 技能达到350级为Level 1
  • 技能达到500级为Level 2
  • 技能达到650级为Level 3

# 刀法绝招【五岳华斩】

指令：perform wuyuehuazhan

条件：

  • 基本刀法、任意四种刀法技能均达到350级
  • 古战场完成绝招领悟

消耗：

  • 内力：200×LEVEL
  • busy时间：0～1秒
  • CD时间：5秒

效果：

  1. 每刀造成气血软硬伤害
  2. 再根据命中数造成爆炸，产生百分比气血硬伤害

影响：

  • 基础命中率取决于平均刀法级别、对方基本轻功、基本格挡以及双方武学
  • 对方处于busy、no_move、晕倒状态时，提升命中
  • 刀气硬伤害：有护具（帽、鞋、衣、手、颈、盾、甲）时为25%，无护具时为50～100%

技能级别与效果：

  • 半月斩（LV1）：命中率46%；命中即爆炸，产生对方当时气血1/3的硬伤害
  • 三圣华斩（LV2）：三击命中率分别为55%、62%、67%；命中刀数n达到2即爆炸，产生对方当时气血n/(1+n*2)的硬伤害
  • 五岳华斩（LV3）：五击命中率分别为65%、71%、75%、78%、80%；命中刀数n达到3即爆炸，产生对方当时气血n/(1+n*2)的硬伤害

# 剑法绝招【神剑天雷】

指令：perform shenjiantianlei

条件：

  • 基本剑法、任意四种剑法技能均达到350级
  • 道家仙法/八卦咒/月宫仙法/太乙仙法/九转玄功/五雷仙法其中之一达到250级
  • 在天雷台完成绝招领悟

消耗：

  • 法力：100+enchant值
  • busy时间：0～1秒
  • CD时间：5秒

效果：

  • 惊雷闪（LV1）：电系气血软伤害，1/2气血硬伤害
  • 五雷轰顶（LV2）：电系气血精神软伤害，2/3气血精神硬伤害
  • 狂雷天牢（LV3）：电系气血精神软伤害，3/4气血精神硬伤害，3秒no_move no_pfm

因素：

  • 成功率：受平均剑法、法术（上述道家法术中最高的）、LEVEL、双方精神、对方轻功影响。对方昏迷、busy、no_move时成功率提升。
  • 伤害：受enchant、平均剑法、法术（上述道家法术中最高的）影响。伤害=基础伤害×系数，L1系数12/6，L2系数12/7，L3系数12/8。

# 拳法绝招【只手遮天】

指令：perform zhishouzhetian

条件：

  • 扑击格斗之技、任意四种拳法均达到350级
  • 房间内没有遮天之手
  • 经拳皇历练完成绝招领悟

消耗：

  • 内力：500
  • 若第一段遮天攻击失败，则自身产生0～1秒busy
  • CD时间：与打到的敌人数有关，最少5秒，最高11秒

效果：

  1. 对战斗中的敌人发出一只遮天巨手。成功则房间内敌人无法使用任何命令且房间产生「遮天蔽日」效果。LV1持续3秒，LV2、LV3持续6秒。L3额外对敌产生3秒busy。
  2. 对不多于10个敌人造成暗系气血软伤害和1/2气血硬伤害。

因素：

  1. 成功率与最大内力、拳法平均级别、武学和在场敌人中武学、轻功最高值有关。若敌人处于busy、no_move、昏迷状态，则成功率提升。
  2. 敌人先天定力高于拳法平均级别的1/4～1/2，或昏迷及隐身可以躲避伤害。伤害与最大内力、拳法平均级别、武器（拳套）攻击力和敌人最大内力有关。

# 叉法绝招【大海无量】

指令：perform dahaiwuliang

条件：

  • 基本叉法、任意四种叉法均达到350级
  • 在竹筏上完成绝招领悟

消耗：

  • 内力：500
  • CD时间：与打到的敌人数有关，最少5秒，最高11秒

效果：

  • 对战斗中的敌人进行攻击并产生debuff，影响敌人不超过10个。叉法平均级别L为效果因数。
    • 对敌人进行一次攻击，造成精神软伤害，伤害值D=L+武器伤害力。LV2时额外造成D/3精神硬伤害。LV3时额外造成D/3气血硬伤害。
    • 使敌人陷入no_exert或no_cast状态，持续L/50秒，最短10秒。同时，使敌人陷入no_move或no_perform状态，持续L/100秒，最短5秒。

因素：

  • 成功率与L和敌人有效轻功等级的随机数有关。
  • 无法对昏迷和不可见敌人产生伤害和效果。
