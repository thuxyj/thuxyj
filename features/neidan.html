# 内丹系统

# 简介

大凡修炼，无论仙、佛、妖，都上体天心，参玄悟道，修炼道行；又淬炼肉身，打磨心神。修行入门后，凝聚周身精、气、神，可化玄丹。道家称内丹、佛家称舍利、妖类形成妖丹。丹成后，乃避三灾，则可成大道，不生不灭。为便于表述，统称内丹。

# 内丹分类

各门各派，修炼之法不一而足，大抵可分仙、妖、佛三流。按所修习的法术，可凝结不同类型的内丹。

  • 仙：道家仙法、八卦咒、太乙仙法、九转玄功、月宫仙法、五雷仙法
  • 妖：勾魂术、碧海神通、登仙大法、平天大法、盘丝大法、逍遥大法、妖法、天魔大法（方寸青狮老魔）
  • 佛：大乘佛法、密宗佛法、孔雀明王经、小乘佛法

不同法术系别，凝聚不同形态的内丹。修炼大成后，还可转换为二阶。若先后兼修多门法术，则可转换内丹为其他种类。

# 内丹凝结

玩家道行20年，内力法力均达到500后，打坐冥思之时便会提示已形成内息。此后用score查看，可看到内视：内息，便可凝结内丹。

输入neidan generate即可凝聚内丹，形成的内丹形态，仅与凝聚时enable的法术有关，要求该法术50级。

# 内丹转换

若想修炼兼修型内丹，则需要在凝聚内丹后再次使用neidan generate命令，此时若所enable的法术系别与内丹类型不同，会使其转换形态。

例如：你已凝聚舍利，若使用道家仙法再次凝聚，将转换为舍利子内丹，若此后再使用妖法凝聚一次，则将转换为神魔舍利。

条件：

  • 第一次转换要求百年道行、消耗十年道行，所使用法术100级
  • 第二次转换要求三百年道行、消耗三十年道行，所使用法术150级
  • 已经二阶的内丹无法转换形态

注意：以下功能使用时均要求内丹形态与当前法术兼容！

如果最终使用的法术是佛，那么可以选择二阶内丹有：

  • 金身舍利：直接用佛系法术生成舍利，再升二阶就可以
  • 金丹舍利：读太乙仙法50级，激发太乙，生成内丹，再激发佛系法术，再次生成舍利子内丹，再升二阶
  • 天魔舍利：读天魔大法50级，激发天魔大法，生成妖丹，再激发佛系法术，再次生成妖舍利，再升二阶
  • 九转神魔舍利：读天魔大法50级，太乙100级，第一次先用天魔大法（激发） 生成妖丹，然后用太乙生成神魔丹，再转回佛系法术，生成神魔舍利，再升二阶

TIP

上面的结丹顺序可以换，如：可以要先用妖系生成妖丹，转佛系再次生成妖舍利，也可以先用佛系生成舍利，转妖系再次生成妖舍利

# 内丹升级

玩家修炼内丹后，便会触发天劫机制，关于天劫的介绍详见天劫。成功渡过三次天劫后，内丹会自动转换为二阶。若玩家选择炼制渡劫丹躲避天劫，则内丹未经天劫淬炼，却需消耗道行自行升级。

  • 主动升级条件：
    • 道行两千年，内力法力最大值2500
    • 所用法术与内丹形态兼容且250级
  • 消耗：一百年道行
  • 命令：neidan levelup

内丹升级为二阶后，会根据内丹形态，给玩家增加一条新的外观描述。

# 内丹重炼

玩家内丹修炼到二阶后，会根据内丹形态获得一项专属功效。然而这一功效随着玩家选择的修炼方式、门派变化，以及玩家对内丹功效的深入理解，可能会有不同需求。此时玩家可以选择将内丹重炼为新的形态。

  • 命令：neidan rebuild <新形态名>
  • 要求：内丹必须已达到二阶，新的形态必须是玩家法术支持的形态

若玩家已升级了原有extra功效，则转换后该功效会自动转换为新形态对应功效，因此玩家法术等级必须满足新功效对等级的要求。重炼的内丹也会产生相应的外观描述。例如，玩家使用大乘佛法，原本修炼了金身舍利，产生特效extra_spirit不坏金身并修炼到了9级。neidan rebuild 天魔舍利，则内丹转换为天魔舍利，特效转换为9级extra_anger金刚怒相。

  • 消耗：第一次重炼消耗百年道行，每重炼一次，消耗增加百年道行

# 内丹修炼

不同类型的丹除有共同的强化、 修补自身之功效，也分别有其独特作用。除初始凝结和内丹晋升二阶自带的功效以外，均需玩家自行修炼及升级。二阶内丹所自带的特殊效果无法用raising修炼产生。

功效分一阶和二阶，二阶功效需要二阶内丹方可修炼和升级。所有功效最高20级，分别显示为一重。。九重、大成、一品。。。九品、圆满。

# 内丹修炼度

内丹修炼度 = 一阶功效级数累加 + 二阶功效级数累加*2

修炼度达到100后，显示为"百炼"内丹，此后方可修炼大成以上级别。特殊：斡旋造化功效需要内丹"百炼"方可开始修炼

功效产生使用neidan raising xxx，升级前有提示。修炼新的内丹功效需要消耗武学，随着已有功效数递增。

# 功效升级

升级已有功效条件及消耗与功效所属系列有关，法术精神系需要并消耗道行、法力，武功气血系需要并消耗武学、内力，特殊系全消耗。

一阶功效升级条件：（n=已修炼级别）

  • 要求：经验（武学/道行）200kn，内力/法力200n，当前高级法术20*n，所使用的高级法术与当前内丹类型兼容
  • 消耗：经验1kn，内力/法力2n
  • 命令：neidan upgrade xxx，请先用neidan cost查看已有功效升级消耗

二阶功效条件及消耗是一阶功效的二倍

# 内丹使用

  • 内丹功效均有被动功效和主动功效，其中主动功效和部分被动功效有CD时间
  • 内丹功效无需充能，随属性升级，减少使用CD、增强效果
  • 内丹使用要求当前激发法术与内丹形态兼容，使用方式为 neidan use <功效>
  • 功效使用或触发后产生的CD会在使用neidan命令查看内丹状态时显示在"蛰伏"栏下

# 不坏金身 extra_spirit

召唤的护法属性、经验复制主人，技能为主人的高级法术技能的(95+n)%级。其中parry级别翻倍（提高guard成功率），金刚的气血精神量也是主人的二倍。召唤的金刚会与主人进行精神、气血的连接（lifeheal），使用普陀功夫，会cast bighammer、pfm diyu、pfm storm，战斗中随机治疗主人精神气血。

# 造化金丹 extra_dan

使用造化金丹主动功效，或服用金丹，或使用其他带有恢复满状态类效果时，会产生一个"恢复术"buff，每秒治疗1气血精神、恢复2气血精神，但在此buff期间内，无法再次使用如上恢复满状态的效果。简言之，所有恢复满状态类的道具、功效有一个共CD效果。extra_dan的主动功效使用时，产生的恢复术buff时间为360秒。

# 相关命令

内丹系列命令为 neidan <子命令>，用help neidan查看详细信息
指令格式 : neidan [<命令内容>]
此命令用于修炼和使用内丹。

命令格式命令作用
neidan内视察看内丹状态
neidan generate凝结内丹或内丹转型
neidan levelup消耗自身修为，强行提升内丹品阶
需2000年道行，消耗100年
neidan rebuild <形态名>重炼二阶内丹，变为对应名称的形态
neidan list查询内丹功效
neidan cost查询功效升级需求
neidan raising <功效>获得一个新功效
neidan upgrade <功效>修炼一个已有功效
neidan use <功效>使用功效的主动效果
neidan add <属性>使用斡旋造化属性点改造自身
