# 法宝系统

# 〖法宝概述〗

西游记里有三大类不同的法宝：
第一类是由仙人道人或者妖魔鬼怪修炼千百年的神物宝贝，玩家通常可
以通过解迷、杀死ＮＰＣ(非玩家角色)等方式找到或获得这类法宝。
这些法宝有些可以装备起来作为武器防具，有些可以通过祭（ji）的方
式使用其特殊效果。比如青云宝剑的特殊攻击、紫金红葫芦和羊脂玉净瓶
的装人效果、捆仙绳和缚龙索的绑人效果，等等。
此类法宝具有唯一特性，一旦被玩家拥有，再掉落的就是假货。假货的
性能要差很多，并且没有专属特效。用checkunique 法宝名称 可以查询
法宝的拥有情况(不加参数列出法宝清单，参数all列出所有法宝状态)

第二类是玩家在法宝炼制场制造自己的法宝，分武器和护具两种不同的
法宝。玩家炼制法宝需要有相当的道行，法宝炼制之后即可以随身携带
象武器和护具一样使用，法宝可以不断地升级，以加强其伤害力，防御
力和抵抗力等。

第三类法宝是隐藏在特定的场所（副本）中，玩家杀死NPC随机掉落，
此类法宝属性随机性比较强，也会产生极其强大的知名法宝原型乃至上古
仙魔的法宝。

以下说明均针对自制法宝。

# 〖基本指令〗

自制法宝需到玉女峰顶女娲补天处进行，制造法宝道行须高于二十年，
法力修为腾云驾雾，内力修为三年以上方可。若法宝丢失或损毁，可到
补天补天密洞使用pray命令恢复。若无法宝丢失而pray，会遭受惩罚！

# 女娲补天处相关指令：

炼制法宝 make_fabao
注销法宝 dispose
法宝检查 look <法宝id>
增加特性 raising <法宝id> <特性>
升级特性 upgrade <法宝id> for <特性>
特性列表 list

自定义项 alter <法宝id> <自定义项目> <自定义内容>
自定义项目可在补天处输入alter查询

# 法宝自带指令：

法宝充能 charge <法宝id> for <特性> [次数]
特性使用 feel <法宝id> for <特性>
法宝拾取 pick <法宝id>

# 〖特性概述〗

除法宝自带初始特性外，每增加一个特性（raising）都需消耗武学。
根据法宝已有特性种类的数量n，消耗量=(n*(n+1)/2+1)*100k

已有2345678910
消耗0.4M0.7M1.1M1.6M2.2M2.9M3.7M4.6M5.6M

法宝特性除了具备天然的被动效果外，法宝特性具备能量储存功能，
可进行充能（charge）后，需要时主动或被动触发其能量，以实现其
额外功能。每个特性初始具备1格能量槽，升级（upgrade）来增加，
最多50格。升级到25格以上，需要在充能到满时观察其状态，当法宝
跳动时方可升级。跳动几率随着格子数增加而降低。

# 升级

法宝升级需要达到一定道行和内力|法力，并损失部分道行和内力|法力
上限，具体用cost查询。格子红色损耗内力，蓝色损耗法力，其他颜色
损耗两者。
以下全文M代表特性已升级格子总数，N代表已充能格数。
升级所需条件及消耗基数（道行单位：年） 法宝最初自带特性的被动效果不计算第一个格子

消耗 = (M+1)*消耗基数

# 充能

充能消耗的能量种类与升级相同，每充1格消耗100点内力（法力）。
充能次数最大为50，若输入的次数超过需要充能的格子数，则自动充满。未充能的格子显示为□，充能后为■

# 使用

有些特性会在战斗中造成伤害或被伤害时自动生效，有些则在满足特定条件时生效，
还有些需要主动使用（feel）来产生额外效果。
主动或被动效果均判定生效率=M*2%，判定失败则无效。
自动触发的效果消耗：(消耗值大于N/2时，消耗N/2点)
M值 <10 <20 <30 <40 <50 50
消耗 1 2 3 4 5 1
主动feel的效果消耗量见下文。
以下特性不受触发几率限制，100%触发（守城等特殊场景限制除外）：
武器法宝的ji功能；防具法宝抵抗ji的功能；extra_trap的穿梭功能；喂毒效果；

# 〖祭法宝〗

法宝除了可以装备起来作为武器和防具使用，还可以祭（ji）来发挥特效。
武器法宝的attack_qi、attack_shen效果会起到特效攻击作用；
防具法宝的defense_shou、extra_catch、extra_trap会起到辅助性特效作用。
祭法宝后自身产生1~3秒busy
命中效果详见特性功能描述。
祭法宝成功需要进行三轮攻防比拼，全部成功则产生效果。
1、道行随机数对比，同等水平有50%成功率；
2、攻击方道行+防御方武学/3的随机数与防御方武学/3对比，同等水平有75%成功率；
3、法力随机数对比，同等水平有50%成功率，如果法力对比失败则进行法宝比拼：
若防御方拥有已装备的防具法宝，比较如下法宝特性的M值，
进攻方所祭法宝的attack_qi、attack_shen、extra_attack（武器法宝）
或 defense_shou、extra_trap、extra_catch（防具法宝）
与防御方防具法宝的defense_shou、extra_trap、extra_catch
进行随机数比拼，差距大时输的法宝对应三特性消耗全部N值
防御方如有两个防具法宝，则采用接力车轮战的方式参与比拼

# 〖特性功能〗

下文气血|精神对应名称定义：

气血： 气血值/ 气血上限 (80%)     气血： 气血值/ 气血最大值 (100%)
精神： 精神值/ 精神上限 (80%)     精神： 精神值/ 精神最大值 (100%)

# attack_qi

# attack_shen

对敌人ji法宝命中后，造成法系气血软伤害和1/2硬伤害（可被defense_qi抵消）
基础伤害值=N*后天灵性+法力enchant值-敌人法力enchant值/2-敌人法力enchant值/2的随机数，消耗N/2格子

# defense_qi

# defense_shen

受到法术攻击时自动触发抵消伤害
当伤害D>=20N时，伤害降低20N点，消耗N/2格
否则伤害完全抵消，消耗(D/20+1)/2格

# defense_shou

  1. 被敌人ji法宝攻击时，参与法宝比拼，详见〖祭法宝〗
  2. 被敌人使用产生busy（有时体现为no_move）的特效攻击产生抵抗效果
    （对debuff类和自己使用技能法术导致的busy和no_move不产生作用）
    初始busy时间为T，抵抗效果如下：
  • 若T*5>=N，则T=T-N/5，消耗N/2格
  • 否则T=0（即完全抵抗），消耗(T*5+1)/2格

# extra_catch

  1. 通过feel使用，产生服用金丹效果，治疗并回复N*2%气血精神和N*4%内力法力，并获得一个恢复术buff，持续100-M秒，每秒治疗精神气血N点，恢复精神气血2N点。与内丹extra_dan效果、服用金丹后产生的buff共用，期间无法使用同类特效。成功使用则消耗1格。busy时可用，特殊地点不可用
    注：恢复术效果均只能恢复到满值而非二倍
  2. 通过ji使用，可以将对方装进法宝中，敌人精神值降为当前精神值的(100-N)%
    同时导致对方昏迷。装人时间T=10*(25+N-敌人后天福源随机数)。同时只能装一个人。
    时间满则自动释放并唤醒敌人，期间敌人若离开法宝，则自动苏醒并busy 5秒

# extra_trap

  1. 通过feel 法宝id extra_trap使用，脱离战斗并穿梭到自己设定的开始点。成功使用则消耗1格。busy时可用，特殊地点不可用
  2. 通过ji使用，将对方捆住，敌人气血值降为当前气血值的(100-N)%
    同时附加no_move、no_pfm效果，持续时间T=10*(25+N-敌人后天福源随机数)
    T最低60，敌人为玩家时减半，并可被敌人已装备法宝的defense_shou属性抵消
    消耗一半格子，持续时间满后释放，此前不能再捆人

以下feel产生的效果均自动计算恢复尽量多的值，直至耗光格子

# extra_attack

武器法宝作为主手武器命中敌人且几率判定成功后，造成精神|气血硬伤害
伤害值=敌人当前精神|气血上限*(N+50)/300

# damage_qi

# damage_shen

武器法宝发出普攻|普攻类pfm|叛师大招时产生额外攻击，造成法系气血软硬伤害
基础伤害值=M*N/5+M/15*自身气血最大值/100 （M/15单独取整计算）
几率判定成功则消耗1格，伤害值需经敌方已装备防具的armor_qi属性修正

# meteo_iron

武器法宝命中敌人时产生敌人产生no_perform效果
几率为每格增加1%，buff初始时长1秒，每升级10格(包括初始格)增加1秒
几率判定成功则随机消耗1~2格

# armor_qi

# armor_shen

  1. 受到来自damage_qi的伤害：
    • 若基础伤害值D>N，则伤害值-N，消耗N/2格
    • 否则伤害完全抵消，消耗(D+1)/2格
  2. 受到普通伤害D点
    • 伤害抵消值R=10N
    • 若D>R，当D>R+被攻击者气血最大值/20，则D降低（R+被攻击者气血最大值/20），消耗N*2/3格；否则伤害完全抵消，消耗N/2格
    • 若D<=R，则伤害完全抵消，消耗格子数=(D/2+2)/3

# recover_qi

# recover_shen

  1. 通过feel使用，每格可恢复2%气血值，最多恢复到当前气血上限
  2. 气血值<0时自动触发恢复气血，格子消耗N*4/5，治疗量=气血最大值/100*N

# heal_qi

# heal_shen

  1. 通过feel使用，每格可治疗1%气血上限，最多治疗到气血最大值
  2. 气血上限<0时自动触发治疗气血上限，格子消耗N*9/10，治疗量=气血最大值/200*N

# recover_neili

# recover_fali

  1. 通过feel使用，每格可恢复100点内力，最多恢复到二倍内力最大值
  2. 内力低于最大内力/5时自动触发恢复内力，消耗1格，恢复量=最大内力/100，每次最多恢复100点
  3. 内力不足二倍最大内力且N大于500的随机数（50格有1/10几率）时自动触发，恢复内力量=N/5，不消耗格子

# recover_food

# recover_water

  1. 通过feel使用，每格可恢复100点食物，最多恢复满
  2. 食物低于1点自动触发恢复N点食物，消耗1格
